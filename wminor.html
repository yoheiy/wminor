NAME

wminor - GID-friendly window manager

SYNOPSIS

wminor

DESCRIPTION

wminor is a minimalistic window manager. wminor is a non-reparenting window manager. Each window has titlebar. Titlebars are only UI wminor provides. Window move is restricted to the screen real estate.

Keyboard bindings

wminor has no keybind. Please try ratpoison(1) to manipulate windows with keyboard.

GID button bindings

Drag titlebar with

Button1 (left button)
to move window,
Button2 (middle button)
to move window in restricted way, and
Button3 (right button)
to resize window.

BUGS

wminor is in alpha state, there are many small bugs.

  • Work well with upto 80 client windows.
  • No support of ICCCM and EWMH.
  • No support of iconified and maximized window.
